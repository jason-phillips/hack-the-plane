# DDS Hack the Plane

Hack the Plane with [Legos](https://www.lego.com/en-us) and [Arduinos](https://www.arduino.cc/)!

We'll need help with the following:
1. Building the kits
2. Programming the Arduino devices
3. Assembling the custom made boards.
